# Dooray Messanger 설치

접속 : https://dooray.com/main/download/ 

macOS용 다운로드 후 다운로드 폴더에서 .dmg 파일 더블 클릭

![메신저다운](./images/04.png)

<br/>

- 모바일 기기에도 설치
    - 안드로이드 : 플레이스토어 `Dooray!` 검색 후 다운로드
    - 아이폰 : 앱스토어에 `Dooray!` 검색 후 다운로드
